# Expense Tracker

A simple Expense Tracker application built using React and Vite.

## Features
- Add expenses
- Track spending
- Simple and responsive UI

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Installation

Clone the repository:

git clone https://github.com/MohammedArif1245/Expense-Tracker.git

Install dependencies:

npm install

Run the project:

npm run dev
